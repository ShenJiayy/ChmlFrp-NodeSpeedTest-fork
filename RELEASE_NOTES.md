# ChmlFrp 节点推荐器

ChmlFrp 节点推荐器正式发布！这款工具专为 ChmlFrp 内网穿透用户打造，聚焦节点筛选、推荐核心需求，助力大家更高效地选择适配的节点，大幅提升内网穿透使用体验。

## 🌟 核心功能

### 🔍 节点自动探测
- 支持调用 API 扫描 ChmlFrp 可用节点列表
- 快速获取节点的基础信息（如节点地区、运营商、在线状态）
- 无需手动查询，一键获取最新节点列表

### 🎯 多维度筛选
- 提供 VIP 节点、国内/国外、UDP 支持等筛选条件
- 用户可根据自身网络环境和需求，精准定位符合要求的节点
- 提高筛选效率，节省时间

### ⚡ 节点测速评估
- 内置 TCP 连接测速功能
- 对筛选后的节点进行延迟检测
- 直观展示本地到节点的连接质量
- 支持按延迟或编号智能排序

### 💾 历史记录保存
- 自动缓存测试结果
- 刷新页面或切换界面后数据不丢失
- 方便后续对比使用，避免重复测试

### 📌 窗口置顶
- 支持窗口置顶功能
- 方便在正常使用电脑时随时查看测试进度和结果

### 🔄 自动更新
- 支持 GitHub 检查更新
- 新版本发布后可一键下载更新

## 🛠️ 技术特性

### 跨平台支持
- 基于 Tauri + Rust 构建
- 支持 Windows、macOS、Linux 三大主流操作系统

### 轻量化设计
- 占用系统资源少
- 不影响内网穿透主程序的运行效率

### 开源免费
- 完全开源，采用 MIT 许可证
- 可放心使用和二次开发

## 📥 下载安装

### Windows
- 下载 `.msi` 或 `.exe` 安装包
- 双击安装即可

### macOS
- 下载 `.dmg` 文件
- 拖拽到应用程序文件夹

### Linux
- 下载 `.deb` 或 `.AppImage`
- 根据发行版安装

## 📋 系统要求

- **Windows**: Windows 10/11 (x64)
- **macOS**: macOS 11.0+ (Apple Silicon)
- **Linux**: Ubuntu 20.04+ (x64)
